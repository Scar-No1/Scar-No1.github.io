What is Bitcoin?

Bitcoin is a cryptocurrency invented in 2008 by an unknown person called Satoshi Nakamoto and started in 2009 when its implementation was released as open-source software. It is a decentralized digital currency without a central bank or single administrator that can be sent from user to user on the peer-to-peer bitcoin network without the need for intermediaries. Transactions are verified by network nodes through cryptography and recorded in a public distributed ledger called a blockchain.

How to obtain Bitcoins?

There are two main method to obtain Bitcoin.

First is using common currency to exchange bitcoin. The exchange rate now is 1 bitcoin for 18980.8 US Dollar.

The second method is mining. Mining is a way which using computer to process a programme, when the result processing out is with in a range, the processer which also called miner can get a reward of bitcoins.

Mining

It is a record-keeping service done through the use of computer processing power. Miners keep block chain consistent, and unalterable by repeatedly grouping newly broadcast transactions into a block, which is then broadcast to the network and verified by recipient nodes. Each block contains a SHA-256 of the previous block, thus linking it to the previous block and giving the blockchain its name.

Early bitcoin miners used GPUs for mining, as they were better suited to the proof-of -work algorithm than CPUs. Later amateurs minded bitcoins with specialized FPGA and ASIC chips. Today, high-performance mining hardware is used.
