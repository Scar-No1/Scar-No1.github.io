ARDUINO Programme 1

Posted on 2020-08-01 | In Computer learning

WHAT IS ARDUINO

Arduino is an open-source hardware and software company, project that provide a platform for people to design their own digital devices.

WHAT WILL WE DO

  • intelligent fan
  • smoke alarm
  • automatic light
  • PDC (PARKING DISTANCE CONTROL)
  • motion recognition
  • manipulator

COMPONENT

MOTHERBOARD

It is the most important unit,which is used for signal collection and output. There are many metal joint on it, they can be divided into three parts.

  • VCC (3.3V/5V)
  • PIN (Analog,Digital)
  • GND

Each joint need to connect to the same type of interface in order to make sure the signal can be successfully transform and prevent the motherboard from burning out due to excessive current.

Other basic originals

  • resistance,luminous diode,wire,bread board

  • keyboard,infrared remote control,buzzer

  • sensor: gray scale sensor,temperature sensor,infrared sensor,smoke sensor

  • sensor: ultrasonic sensor,sound sensor, IMU(inertial measurement unit)

  • motor: servo,direct current motor

  • display: Nixie tube, 1602LCD

ARDUINO Programming Software

It should be noted that the indentation are marked using {}.

All the programming statements can be written through a flow chart.

E.g.
