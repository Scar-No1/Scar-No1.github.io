ARDUINO Programme 3

Posted on 2020-08-12 | In Computer learning

In this class, we learned about how to make an intelligent reversing radar.

Button

From this part, we learned a tip.

In void loop, you have to set the original state of the output first.

digitalWrite(LED,LOW);

Set the state low in order to prevent the fan start to work immediately when the motherboard has just connected to the power.

intelligent reversing radar

Buzzer

This device will make sharp sound when receiving electronic signal.

We build a function called BUZZ.

When there is a time signal input, the buzzer will bee at the speed according to the time period. The larger the time period , the slower the buzzer will bee.

Ultrasonic sensor

This device is used to detect the distance of the object in front of the sensor.

We can use a Serial monitor to show the detailed value.

Final

Combine all these above together.

  • First, the ultrasonic sensor record the time period it emit and receive the signal. Using this time to calculate the distance of the object by multiply the speed of sound wave in air.
  • According to the distance long, send different signal to the buzzer. The smaller the distance, the smaller the time delay of buzzer.(Three stage: 10 cm , 50 cm , 80 cm)
