ARDUINO Programme 2

In the second period, we learned about the theory of automatic light and intelligent fan.

Automatic Light

Infrared

This lamp uses the infrared sensor to test the surrounding. The sensor can detect the objects surrounding with the temperature between 36.0C to 37.4C (Human Body Temperature). If there is, it will transmit the analogue signal to the motherboard to open the light.

  • Infrared sensor

Intelligent fan

Serial

This is a function in Arduino which can help show the running state of the motherboard.

We can use this function in the programming of intelligent fan.

Theory

A temperature sensor is used. It can test the surrounding temperature and send the signal to the motherboard. You can check the detail in the Series Monitor. In the programme , set a standard temperature. If the temperature tested is higher than the set value, the fan will start to work automatically.

Code

Here, I set 36C as a standard temperature. So, when the temperature exceeds 36C, the fan will start.
