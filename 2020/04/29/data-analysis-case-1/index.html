Annotation

[1]: ha is the horizontal alignment

[2]: xy is The annotated position

​ formate:(float, float)

​ The point (x, y) to annotate.

[3]:xycoords is the coordinate system for xy

[4]:xytext:formate:(float, float), optional

​ The position (x, y) to place the text at. If None, defaults to xy

[5]:The coordinate system that xytext is given in.

All xycoords values are valid as well as the following strings:

Value Description
‘offset points’ Offset (in points) from the xy value
‘offset pixels’ Offset (in pixels) from the xy value
